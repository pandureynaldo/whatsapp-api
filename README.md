
### How to use?
- Install Node.js V 14++
- Install nodemon globally
- Clone or download this repo
- Enter to the project directory
- Run npm install
- Run npm run start
- Open browser and go to address http://localhost:7075
- Scan the QR Code
- Enjoy!

### Base Library
- https://github.com/pedroslopez/whatsapp-web.js
